# ComparAR - Proyecto de Grado

Repositorio del proyecto **ComparAR**, plataforma web para la comparación de precios de productos de supermercados.

La aplicación fue desarrollada como prototipo funcional para el Proyecto de Grado de la carrera Ingeniería en Informática.

---

## Aplicación desplegada

La versión online puede probarse desde el siguiente enlace:

https://comparar-proyecto-grado.vercel.app

---

## Instructivos de prueba

Este repositorio incluye tres instructivos principales para la evaluación del sistema.

### 1. Prueba online

Archivo:

```txt
INSTRUCTIVO_PRUEBA_ONLINE.md
```

Contiene los pasos para probar la aplicación ya desplegada, sin necesidad de instalar dependencias ni configurar un entorno local.

---

### 2. Ejecución local

Archivo:

```txt
INSTRUCTIVO_EJECUCION_LOCAL.md
```

Contiene los pasos para ejecutar el backend y el frontend de forma local.

---

### 3. Ejecución de scrapers

Archivo:

```txt
INSTRUCTIVO_SCRAPERS.md
```

Contiene los pasos para ejecutar los scripts de scraping utilizados para actualizar productos, precios, promociones, imágenes y enlaces.
